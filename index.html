Aprendizados

Nessa "primeira etapa" do curso da trybe relacionado a programação, estudei html e css. Com esses conhecimentos foi possível desenvolver esse projeto e fazendo uma introdução ao desenvolvimento frontend. Aprendi a desenvolver a estrutura da página com HTML e a "estilizar" a página e seus elementos com o CSS.

A Mão que ajuda

Um facilitador para o meu aprendizado sobre HTML e CSS foi o site da w3schools com exemplos simples e bem didáticos.

Batendo a Cabeça

Algumas dúvidas que surgiram durante o desenvolvimento desse projeto foram o uso de sections, id vs class , redimensionamento dinâmico sem o uso de javascript, elementos in-line e block e a manipulação das "caixas" .
