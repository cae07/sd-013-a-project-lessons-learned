O que aprendi até o momento na Trybe?

Meu primeiro contato com a programação foi iniciando o processo seletivo para virar aluno da Trybe. Encarei esse desafio em busca de novas oportunidades e consequentemente em busca de uma carreia de sucesso.

Logo nas primeiras semanas aprendemos sobre Unix&Bash, Git&GitHub, HTML&CSS. Com o Git aprendemos a armazenar nossos projetos e exercicios no GitHub, com HTML aprendemos como criar toda estrutura de uma página web e utilizando o CSS para estilizar a página.

Sobre meu aprendizado

O que mais gostei de aprender até o momento foi sobre Git&GitHub, HTML&CSS. No começo tive um pouco de dificuldade para entender o conteúdo mas consegui solucionar todas as dúvidas nos plantões e com nossos amigos da Tribo.
