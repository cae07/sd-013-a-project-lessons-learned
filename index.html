O que aprendi até o momento na Trybe?

Meu primeiro contato com a programação foi iniciando o processo seletivo para virar aluno da Trybe. Encarei esse desafio em busca de novas oportunidades e consequentemente em busca de uma carreia de sucesso.

Logo nas primeiras semanas aprendemos sobre Unix&Bash, Git&GitHub, HTML&CSS. Com o Git aprendemos a armazenar nossos projetos e exercicios no GitHub, com HTML aprendemos como criar toda estrutura de uma página web e utilizando o CSS para estilizar a página. estou muito feliz em reconhecer a evolução que a Trybe tem me dado a cada aula assistida.
